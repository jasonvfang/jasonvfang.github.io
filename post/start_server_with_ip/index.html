Hugo启用自定义IP绑定

Posted on 12 August, 2021 at 13:32 CST by Author Name

Category: programming

默认hugo启用server时会绑定在localhost(127.0.0.1)上，在外网上无法通过IP访问，只能通过本机输入localhost访问， 那么如何启用Server可用于IP访问：


hugo server -b http://10.240.218.181 --bind="0.0.0.0" -D
